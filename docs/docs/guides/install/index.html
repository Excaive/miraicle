环境搭建

安装 miraicle#

和很多其他的 Python 第三方库一样，你可以从 PyPi 安装 miraicle，这需要你的 Python 版本在 3.6 或以上。

pip install miraicle

如果你已经安装了 miraicle，想要把它更新到最新版本，你可以输入：

pip install --upgrade miraicle

安装和配置 mirai-api-http#

miraicle 是基于 mirai-api-http 的，mirai-api-http 又是 mirai-console 的一个插件。在使用 miraicle 之前，请按照 mirai-api-http 的文档进行环境搭建和插件配置。

你可以使用 mirai-console-loader ，它会对 mirai-console 进行一键启动和自动更新。安装好 mirai-api-http 之后，将 mirai-api-http 的 setting.yml 模板复制粘贴到配置文件里，并自己设置一个 verifyKey 和 port。

info

mirai-api-http 目前已经更新到 2.x 版本，这与 1.x 版本相比有不少变动。miraicle 仅支持 2.x 版本的 mirai-api-http ，请检查你的 mirai-api-http 版本是否正确。

简介

众所周知，mirai 是一个在全平台下运行，提供 QQ Android 协议支持的高效率机器人库。

mirai 是用 Kotlin 语言编写的，但它通过 mirai-api-http 插件，支持了更多语言。miraicle 就是 mirai 的众多 Python SDK 中的一种，你可以用它来编写属于自己的 QQ 机器人。
